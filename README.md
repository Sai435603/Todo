## Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/Todo.git
   cd Todo
2. **Install dependencies**
   ```sh
   npm install
3. **Start the server**
   ```sh
    node index.js
4. **Or with nodemon (if installed):**
    ```sh
    nodemon index.js
5. **Open in browser**
  ```sh
http://localhost:3000
